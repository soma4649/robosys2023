# robosys2023
2023年度千葉工業大学ロボットシステム学講義内での課題提出用リポジトリです。


## このリポジトリで使用可能なコマンド

・plus

## plusコマンドの説明

標準入力から読み込んだ数値を足す

## 使用方法

```
$ seq 10 | ./plus
55
```

## コマンドの使用方法
１．以下のコードをhomeディレクトリで実行してください。

```
$ git clone git@github.com:soma4649/robosys2023.git
```
２. 以下のコードでrobosys2023のディレクトリに移動してください。

```
$ cd robosys2023
```
３．robosys2023のディレクトリで使いたいコマンドを入力してください。
